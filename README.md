## Installation

1. Open Visual Studio Code
2. Press `Ctrl+P` / `Cmd+P`
3. Run `ext install frozen-theme`
4. Select your preferred variant:
    - File > Preferences > Color Theme
    - Search for "Frost"
    - Select Arctic, Aurora, Luminous, or Day
